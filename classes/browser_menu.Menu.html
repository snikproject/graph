menuData

  • menuData(): ({ entries: (string | (() => void))[][]; i18n: string; id: string; label: string } | { entries: any[][]; i18n: string; id: undefined; label: string })[]
    • Defined in browser/menu.ts:98

    Creates and returns the menus for the top menu bar. The format is an array of menu elements. Each menu element is an object with a "label", unique "id" and an "entries" array. entries is an array of arrays of size two. entries[i][0] is either a link as a string (will be opened on another tab) or a function that will be executed. entries[i][1] is a label as a string.

    Returns ({ entries: (string | (() => void))[][]; i18n: string; id: string; label: string } | { entries: any[][]; i18n: string; id: undefined; label: string })[]

    the array of menu elements.
